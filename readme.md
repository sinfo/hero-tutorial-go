## go

```
# build
$ go build -o main *.go

# test
$ go test ./...

# run linter
$ golint ./...

#run
$ ./main
```

## docker

```
# build and run
$ docker-compose up --build
```

## dependencies

- [gorilla/mux](https://github.com/gorilla/mux)
- [dep](https://github.com/golang/dep)
